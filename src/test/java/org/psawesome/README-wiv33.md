[게임방식](https://namu.wiki/w/%EC%88%AB%EC%9E%90%EC%95%BC%EA%B5%AC)

게임 룰
- 0-9 서로 다른 숫자
- 숫자는 맞지만 위치 가틀렸을 때 '볼'
- 숫자, 위치가 맞을 경우 '스트라이크'
- 숫자와 위치가 맞지 않으면 '아웃'


    3 스트라이크일 경우 정답


## 기능 요구사항

* 숫자 
    - [1-9]{3}

* 출력 내용
    - {count} 스트라이크
    - {count} 볼
    - {count} nothing

* 컴퓨터가 가지고 있는 3개의 숫자를 사람이 입력하는 방식

* 숫자 3개를 맞히면 게임 종료

* 게임 종류 후 재시작하거나 프로그램을 종료할 수 있다.
